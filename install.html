Install from update site (recommended)

The latest stable version can always be installed from this update site:

http://github.eclipsesource.com/jshint-eclipse/updates/

  • In the Eclipse main menu, click: Help > Install New Softwareâ€¦
  • Enter the URL above and select JSHint Eclipse Integration

Update

If installed as described above, Eclipse can look for updates by itself:

  • In the Eclipse main menu, click: Help > Check for Updates

Uninstall

The feature can be unstalled as follows:

  • In the Eclipse main menu, click: Help > About Eclipse > Installation Details
  • Select JSHint Eclipse Integration, press Uninstall

Getting started

  • On a project that contains JavaScript files, open the context menu and select Properties
  • On the JSHint page, enable JSHint for at least one file or folder

Download as zip

Zip files can be downloaded on the release page . These zips contain a complete p2 repository and can be used for installation into Eclipse.
