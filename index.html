jshint-eclipse

issues source

JSHint integration for the Eclipse IDE

Install

The latest stable version can be installed from this update site:

http://ralfstx.github.com/update/jshint-eclipse/

  • In the Eclipse main menu: Help > Install New Software...

Use

  • Open the context menu on a project and select Properties
  • On the JSHint page, enable JSHint for this project
  • Exclude non-source files and folders from JSHint validation in the context menu of the respective files/folders
